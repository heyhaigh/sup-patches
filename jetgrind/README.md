# Jet Grind Tag

A Jet Grind Radio / Jet Set Radio style graffiti tag generator for SupChat.

## Features

- Generate stylized graffiti tags from any text
- 7 style modes: Fully Random, GGs, Noise Tanks, Poison Jam, Love Shockers, Grunge, Blackwork
- 520+ billion unique combinations in fully random mode
- Rare drops: Gold (5%), Holographic (3%), Diamond (1%)
- Add JSR-style backdrops (10 locations)
- Tag Portfolio & Photo Journal for saving creations
- Background removal via imageclip integration

## Usage

Type any text (max 20 characters) to generate a graffiti tag.

**Commands:**
- `tags` or `portfolio` - View Tag Portfolio
- `photos` or `journal` - View Photo Journal
